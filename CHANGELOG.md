# Changelog

## 1.2.0

* **Parámetros opcionales:** Agregados `textColor` y `backgroundColor` para personalizar los colores del texto y fondo del footer.

## 1.1.0

* **Se cambiaron:** el mail y numero de telefono por unicamnete emojis.

## 1.0.0

* **Lanzamiento inicial (Initial release):** Creación del paquete `quarks_footer`.
* **Diseño Responsivo:** Implementación de un footer en modo oscuro, de una sola línea, adaptable a resoluciones de Web, Desktop y Mobile usando el widget `Wrap`.
* **Identidad Visual:** Integración del isotipo de Quarks Studio con manejo interno de assets.
* **Enlaces de Contacto:** Agregados botones interactivos para enviar correos (`mailto:`) y realizar llamadas telefónicas directamente (`tel:`).
* **Redes Sociales:** Integración de íconos de Instagram, LinkedIn y Web mediante el paquete `font_awesome_flutter`.
* **Navegación:** Implementación de apertura de enlaces externos seguros utilizando el paquete `url_launcher`.
